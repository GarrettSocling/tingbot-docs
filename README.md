# Tingbot Docs

The documentation for Tingbot.

Built using Mkdocs.

## Building

First,

```
pip install mkdocs
```

To run locally, do-

```
mkdocs serve
```

## Deployment

Deployment to docs.tingbot.com is automatic from the master branch, using [Read the Docs](http://readthedocs.org/).
