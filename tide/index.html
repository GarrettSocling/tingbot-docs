Tide

Tide is the IDE for Tingbot.

Download

Mac Windows Linux
Download DMG Download EXE Download DEB or RPM

Once you've installed Tide, check out the examples in the menu at File > Examples, or maybe try the first app guide!

Contributing

Tide is completely open source, built in Javascript using the Electron framework. Interested how it's built or want to get involved? Check out out Github!
